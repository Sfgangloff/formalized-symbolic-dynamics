# Open problems

Folder reserved for formalizations of *open* problems in symbolic dynamics —
problems whose mathematical statement is settled but whose resolution (proof
or counter-example) is currently unknown.

A typical entry would mirror the structure of [`../papers/`](../papers/): each
open problem gets a folder with

- a write-up of the problem statement,
- a formalization plan for its statement (and any partial results we want to
  formalize),
- an implementation checklist,
- and the corresponding Lean sources in `../SymbolicDynamics/`.
